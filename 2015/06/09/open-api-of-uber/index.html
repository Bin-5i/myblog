UBer Open API：

AUTHORIZATION 授权方式
OAuth 2.0 bearer token or server_token

PRODUCT TYPES （产品类型，即汽车）

  1. GET /v1/products 获取当前位置的车辆信息（包括各个级别档次的车）
    参数：latitude，longitude
    返回：

  2. GET /v1/products/{product_id} 获取某辆车的信息
    参数：车辆ID
    返回：

PRICE ESTIMATES （价格预估）

3.GET /v1/estimates/price 根据开始结束点坐标预估价格
参数：start_latitude，start_longitude，end_latitude，end_longitude
返回：各个档次的车型的预估价格

4.GET /v1/estimates/time 根据开始点坐标查询预估到达时间
参数：start_latitude，start_longitude，customer_uuid (optional)，product_id (optional)
返回：

  1. GET /v1/promotions 获取推广奖励
    参数：start_latitude，start_longitude，end_latitude，end_longitude
    返回：

USER ACTIVITY (V1.2)（用户活动记录）

  1. GET /v1.2/history 获取用户历史记录
    参数：offset，limit
    返回:

USER PROFILE(用户档案)
7.GET /v1/me 返回当前用户的基本信息

REQUEST（叫车）

  1. POST /v1/requests 一次叫车申请
